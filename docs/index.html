Deep-Learning Based Noise Reduction & Speech Enhancement

Project landing page and quick-start guide

Overview

This project provides multiple audio enhancement profiles (Enhanced, Ultra, Extreme) for denoising speech, with a Flask-based web UI and DSP pipelines. GitHub Pages hosts this static landing page; live audio processing runs via the Flask server.

Repository: GitHub

Run Locally

Follow the README for exact steps. Typical setup:

# Clone and enter the project
git clone https://github.com/ghnmqdtg/Deep-Learning-Based-Noise-Reduction-and-Speech-Enhancement-System.git
cd Deep-Learning-Based-Noise-Reduction-and-Speech-Enhancement-System

# Create and activate a virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Start the web app (choose one based on README)
python app_production.py
# or
./start.sh

Once running, open http://localhost:5000 and select your enhancement profile.

Features

  • Enhanced: Balanced denoising for general use
  • Ultra: Aggressive multi-stage cleaning
  • Extreme: Gap silencing with volume preservation
  • Dataset tools and optional deep learning training notebook
